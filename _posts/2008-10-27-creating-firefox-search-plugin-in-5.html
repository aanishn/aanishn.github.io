First thing to do, identify the search URI. For this go to the website, do a real search. When the results come up,
copy the URL displayed in the URI bar of the browser. It should look something like below,
http://searchsite.com/blah/blah/?var=search+term
Replace your search term with {searchTerms}, so it should look like
http://searchsite.com/blah/blah/?var={searchTerms}

Next we have to create an icon that will come up by the side of firefox search bar.
1. Get the image, resize it to 16x16, using any of your favourite image editing tools.
2. Convert it into Windows Icon format, you can use ImageMagick on Linux for this purpose,
#convert image.jpg image.ico
3. We have to base64 encode the image so that we can include it in the XML file,
#base64 image.ico
Keep the output of the command somewhere.

Next go to, .mozilla/firefox//searchplugins directory
#cp somefile.xml myplugin.xml

Open myplugin.xml in your favourite text editor, Edit the ShortName, Description to suite your search plugin's.
Replace the content in between <os:Image width="16" height="16">data:image/x-icon;base64, and </os:Image>
with the output of base64 command.

Now go to Url tag, replace the template's value with your customized URI.

Restart firefox, and you are on.....
